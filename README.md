# OpenGL-light-sample
Example of work with polygons and light source

—Сборка и запуск:

-mkdir build

-cd build

-cmake ..

-make

-./main

—Управление:

Реализованы дополнительные возможности:

-Q - вкл./выкл. движение объектов

-Shift - вкл./выкл. ускорение движения

-W/S - вверх/вниз

-A/D -  влево/вправо
