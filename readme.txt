Решетник Николай, 325гр.

—Баллы:
-База - 10 б (см. Детали)
-Непредусмотренный заданием бонус - 2б (см. Управление)
Сумма: 12 баллов

—Сборка и запуск:
-mkdir build
-cd build
-cmake ..
-make
-./main

—Управление:
Реализованы дополнительные возможности:
-Q - вкл./выкл. движение объектов
-Shift - вкл./выкл. ускорение движения
-W/S - вверх/вниз
-A/D -  влево/вправо

—Детали:
3 различных объекта (все с текстурой и двигаются): модель персонажа, деревянные дощечки в форме квадрата и пятиугольника; 1 источник освещения
Использовались сторонние библиотеки:
-model - парсинг объектных файлов
-geometry - для работы с векторами и матрицами
-stb - работа с изображениями
